# 10PM Curfew Creative Strategy & Ops

Thirteen references: six creative-strategy frameworks from Motion (motionapp.com), four adapted from Rob Palmer's direct-response suite (robpalmer.com, CC-BY-4.0), and a UGC production-and-ops playbook. Everything lives in `references/` — load ONLY the file(s) the current task needs.

## The Creation Pipeline

```
1. WRITE      hook-writing.md          → psychologically-driven hooks from product + persona + awareness stage
2. FRAME      hook-tactics.md          → 35+ tactic types; pick or classify the strategic frame
3. NATIVIZE   hook-voice-patterns.md   → swipe file of proven feed-native templates; fill in the brackets
4. JUDGE      hook-evaluator.md        → strict PASS/FAIL grading (~40% pass rate by design); rewrite failures, re-judge
5. CONCEPT    ad-concept-generator.md  → develop the winning hook into a full ad concept / creative brief
6. SCRIPT     ugc-scriptwriter.md      → turn the concept into a filmable UGC creator script
7. PRODUCE    production-and-ops.md    → beat maps, filming, editing, captions, posting, analysis, organic-to-ad
```

## Working Rules

- **Hooks first.** Concepts, scripts, and pages are only as good as the hook. No hook yet? Generate and judge hooks before anything downstream.
- **Judge before shipping.** Final hook lists go through hook-evaluator.md; only present passes (or show the rewrite loop).
- **Every creator-facing deliverable (UGC script, creator brief, shot list) includes the 10PM Curfew Content Creator Guidelines** — attach references/content-creator-guidelines.md verbatim as a "Content Creator Guidelines" section/page of the deliverable. Non-negotiable; this is what creators are held to.
- **UGC scripts ship hook-swap modular:** one body + 3-5 swappable openers (all judged), spanning multiple format archetypes, with triple-hook specs (verbal/visual/text overlay), sound-off design, and creator delivery bullets — see the Advanced Playbook in ugc-scriptwriter.md.
- **Ad copy ships as 3-5 labeled variations** (different strategic angles), passes the meta-direct-response.md final check, and avoids its banned AI tells.
- **Compliance-check health, supplement, and finance copy by default.** For any Meta ad or lander in those categories (Enhanced, LifeMD, Lotus, Atkins, Physician's Choice, etc.), run compliance-checker.md before delivering — write for conversion first, then edit for compliance.
- **Stack with brand skills.** If a brand-context skill applies, load it too — pull product facts, voice, proof points, and compliance guardrails from it and never contradict them.
- **Benchmarks** ("how many creatives should we test", hit rates) → the separate motion-creative-benchmarks-2026 skill.
- **Inputs** (ask only if missing and needed): product/offer, persona + pain, awareness stage, platform/format, funnel position, destination (PDP/advertorial/VSL). Reasonable defaults beat interrogation — state assumptions and proceed.
